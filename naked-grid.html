The example above is the same grid with some minimal window dressing. To isolate each column I added some rounded corners, margins, and a little bit of padding to the H3 tags embedded in each column. Also, the contents of each cell are highlighted for your viewing pleasure.

Why have only halves (1/2), thirds (1/3), and quarters (1/4) as the only column widths to this grid? Easy, This is my starting point and, for many things that I do, it's the best, least "weighty" way to use a grid system in a lot of my projects. Of course, that default usage may change but I can easily change this grid to suit those conditions as needed.
